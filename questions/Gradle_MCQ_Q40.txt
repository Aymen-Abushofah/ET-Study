MCQ – What Gradle Is (Q1–Q40)
All questions with full options and correct answers
Q1. What is Gradle?
A. A programming language
B. A UI framework
C. A build automation tool
D. A database system
Correct Answer: C
Q2. Gradle is mainly used for building:
A. iOS apps
B. Web applications

Correct Answer: D

Correct Answer: C
