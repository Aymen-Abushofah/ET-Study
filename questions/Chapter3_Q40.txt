Flutter vs Other Frameworks – MCQ (Chapter 3) Q1–Q40
Q1. What does “Native Development” mean?
A. Using one codebase for all platforms
B. Using platform-specific languages and tools
C. Using web technologies only
D. Using plugins for hardware access
Correct Answer: B

Q2. Which language is officially used for Android native development today?
A. JavaScript
B. Swift
C. Kotlin
D. Dart
Correct Answer: C

Q3. One main advantage of native apps is:
A. Larger app size
B. Slower performance
C. Direct access to device features
D. One shared codebase
Correct Answer: C

Q4. Why can project management be harder with native development?
A. Lack of UI tools
B. Needing separate teams for each platform
C. No access to plugins
D. Poor performance
Correct Answer: B

Q5. What feature makes Flutter development faster during UI changes?
A. Garbage Collection
B. Hot Reload
C. WebView
D. Compilation
Correct Answer: B

Q6. Flutter apps are usually larger in size because:
A. Dart is slow
B. Flutter uses JavaScript
C. Flutter engine is bundled with the app
D. Plugins increase size only
Correct Answer: C

B. Flutter

C. React Native
